## 📌 Sobre o Projeto

A **Calculadora do Agricultor** é uma aplicação web desenvolvida em parceria entre a **FATEC Matão - Luiz Marchesan** e a empresa **Tatu Marchesan**. O projeto visa revolucionar a agricultura de precisão através de uma ferramenta digital que auxilia agricultores nos cálculos essenciais para:

- ✅ **Otimizar a produção agrícola**
- ✅ **Reduzir desperdícios de recursos**
- ✅ **Melhorar a eficiência operacional**
- ✅ **Facilitar a tomada de decisões no campo**
- ✅ **Promover práticas sustentáveis**

## 🎯 Objetivo

Desenvolver uma calculadora agrícola inteligente baseada em fórmulas validadas pela **Marchesan**, oferecendo aos agricultores uma ferramenta confiável para realizar cálculos precisos em diversas áreas da agricultura, incluindo:

- 🌱 **Distribuição de sementes por metro**
- 💧 **Cálculo de vazão de bicos de pulverização**
- ⚙️ **Eficiência operacional de implementos**
- 📏 **Conversões de unidades agrícolas**
- 📊 **Análise de produtividade**
- 💰 **Cálculos de custos operacionais**

## 🛠️ Instalação e Configuração

### 📋 **Pré-requisitos**

Antes de começar, certifique-se de ter instalado:

- **Node.js** (versão 18.0 ou superior) - [Download](https://nodejs.org/)
- **npm** (incluído com Node.js) ou **yarn**
- **Git** - [Download](https://git-scm.com/)
- **Visual Studio Code** (recomendado) - [Download](https://code.visualstudio.com/)
- **Conta Firebase** - [Criar conta](https://firebase.google.com/)

### 🚀 **Instalação Rápida**

#### 1️⃣ **Clone o Repositório**
```bash
git clone https://github.com/Calculadora-do-Agricultor/calculadora-do-agricultor-front.git
cd calculadora-do-agricultor-front
```

#### 2️⃣ **Instale as Dependências**
```bash
# Usando npm
npm install

# Ou usando yarn
yarn install
```

#### 3️⃣ **Configure o Firebase**

1. **Crie um projeto no Firebase Console**:
   - Acesse [Firebase Console](https://console.firebase.google.com/)
   - Clique em "Adicionar projeto"
   - Siga as instruções de configuração

2. **Configure a autenticação**:
   - No console do Firebase, vá para "Authentication"
   - Ative o método "Email/senha"

3. **Configure o Firestore**:
   - Vá para "Firestore Database"
   - Crie um banco de dados
   - Configure as regras de segurança

4. **Obtenha as credenciais**:
   - Vá para "Configurações do projeto"
   - Na seção "Seus aplicativos", adicione um app web
   - Copie as credenciais de configuração

#### 4️⃣ **Configure as Variáveis de Ambiente**

Crie um arquivo `.env.local` na raiz do projeto:

```env
# Firebase Configuration
VITE_FIREBASE_API_KEY=your_api_key_here
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id

# Optional: Analytics
VITE_FIREBASE_MEASUREMENT_ID=your_measurement_id
```

#### 5️⃣ **Execute o Projeto**

```bash
# Desenvolvimento
npm run dev

# Ou com yarn
yarn dev
```

O aplicativo estará disponível em: `http://localhost:5173` (ou outro endereço parecido)

### 🏗️ **Build para Produção**

```bash
# Build otimizado
npm run build

# Preview do build
npm run preview

# Deploy no Firebase (opcional)
npm install -g firebase-tools
firebase login
firebase deploy
```

### 🔧 **Scripts Disponíveis**

| Script | Descrição |
|--------|-----------|

| `npm run build` | Cria build de produção |

## 📖 Manual de Uso

### 🏠 **Página Inicial**

A página inicial apresenta:
- **Banner principal** com informações sobre a aplicação
- **Seção de funcionalidades** destacando os principais recursos
- **Call-to-action** para cadastro ou login
- **Informações sobre a parceria** FATEC e Marchesan

### 👤 **Sistema de Autenticação**

#### **Cadastro de Usuário**
1. Clique em "Cadastrar" no menu superior
2. Preencha os dados obrigatórios:
   - Nome completo
   - Email válido
   - Senha (mínimo 6 caracteres)
3. Aceite os termos de uso
4. Clique em "Criar Conta"
5. Verifique seu email (se necessário)

#### **Login**
1. Clique em "Entrar" no menu superior
2. Digite seu email e senha
3. Clique em "Entrar"
4. Será redirecionado para a calculadora

#### **Recuperação de Senha**
1. Na tela de login, clique em "Esqueci minha senha"
2. Digite seu email
3. Verifique sua caixa de entrada
4. Siga as instruções do email

### 🧮 **Usando a Calculadora**

#### **Interface Principal**
A calculadora possui:
- **Sidebar esquerda**: Lista de cálculos disponíveis
- **Área central**: Formulário do cálculo selecionado
- **Área de resultados**: Exibição dos resultados em tempo real

#### **Realizando Cálculos**
1. **Selecione um cálculo** na sidebar
2. **Preencha os parâmetros** nos campos do formulário
3. **Visualize os resultados** automaticamente
4. **Copie os resultados** clicando no botão de cópia
5. **Salve o cálculo** (opcional) para histórico

#### **Filtros e Busca**
- **Busca por nome**: Digite no campo de busca
- **Filtro por categoria**: Use o dropdown de categorias
- **Filtro por complexidade**: Selecione o nível desejado
- **Favoritos**: Marque cálculos como favoritos

### 📊 **Gerenciamento de Cálculos**

#### **Histórico de Cálculos**
1. Acesse "Meus Cálculos" no menu
2. Visualize todos os cálculos realizados
3. Filtre por data, categoria ou tipo
4. Exporte dados em diferentes formatos

#### **Criando Cálculos Personalizados** (Admin)
1. Acesse "Criar Cálculo" no menu admin
2. Defina:
   - Nome e descrição
   - Categoria
   - Parâmetros de entrada
   - Fórmula de cálculo
   - Unidades de medida
3. Teste o cálculo
4. Publique para os usuários

### ⚙️ **Configurações**

No menu de configurações você pode:
- **Editar perfil**: Nome, email, telefone
- **Alterar senha**: Segurança da conta
- **Preferências**: Tema, idioma, notificações
- **Privacidade**: Controle de dados e geolocalização
- **Logout**: Sair da aplicação

### 📱 **Uso Mobile**

A aplicação é totalmente responsiva:
- **Menu hambúrguer**: Navegação em dispositivos móveis
- **Gestos touch**: Suporte a toques e deslizes
- **Teclado otimizado**: Campos numéricos abrem teclado numérico
- **Orientação**: Funciona em portrait e landscape

### 🏛️ **Padrões Arquiteturais**

#### **📦 Componentização**
- **Atomic Design**: Componentes organizados hierarquicamente
- **Single Responsibility**: Cada componente tem uma responsabilidade específica
- **Reusabilidade**: Componentes reutilizáveis em diferentes contextos
- **Props Interface**: Interface clara e tipada para props

#### **🔄 Gerenciamento de Estado**
- **Context API**: Para estado global (autenticação, tema)
- **useState**: Para estado local dos componentes
- **Custom Hooks**: Lógica reutilizável encapsulada
- **Firebase Hooks**: Integração reativa com Firebase

#### **🛣️ Roteamento**
- **React Router**: Navegação client-side
- **Rotas Protegidas**: Controle de acesso baseado em autenticação
- **Lazy Loading**: Carregamento sob demanda de páginas
- **Nested Routes**: Rotas aninhadas para melhor organização

#### **🎨 Estilização**
- **Tailwind CSS**: Framework utility-first
- **CSS Modules**: Estilos com escopo local
- **Responsive Design**: Mobile-first approach
- **Design System**: Componentes consistentes
